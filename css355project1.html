How Swift allows you to add tapping/swiping gestures

One of the most appealing functionalities of an iPhone (any phone in general) is to be able to swipe and tap. I remember first using an iPhone and features like this were most interesting to me. When you touch the screen of an iPhone you usually want some function to occur. Swift makes this happen by using something called a gesture-recognizer. This function takes two parameters, the first parameter tells the Swift whats being tapped, and the second parameter is the code you want to run when a user taps.


let gesture = UITapGestureRecognizer(target: self, action: #selector(userTapped))

Core Data (iOS framework to save Data)

Synonyms between OOP, Core Data, and DataBases

Class Entity Table

Core Data would save all the entities from your applications code into a persistant container.We then use a context which acts like a mediator between our application and the persistent container. The context is a temporary area where you can make changes to the database. The changes you can make to the database are CRUD creating more data, reading the data, updating data, and destroying data. Once you're happy with the changes you made with the data, you then can run the code below which will update the changes you made to the persistent container.

try context.save()

This process is similar to git because you first add the changes you wanted to make to your project and then when you're sure about the changes you've made, you then use a commit command when your happy with the changes.
